## 📖 Sobre o Projeto

Dark Runner RPG foi criado com o objetivo de aplicar conceitos de:
- lógica de programação
- desenvolvimento frontend
- animações
- colisões
- inteligência artificial básica
- manipulação de canvas

## 🎮 Controles

| Tecla | Função |
|------|---------|
| W A S D | Movimentação |
| Espaço | Ataque |
| R | Reiniciar |

## ⚡ Funcionalidades

- Sistema de combate em tempo real
- Inimigos com perseguição inteligente
- Boss final
- Múltiplos mapas
- Sistema de colisão
- HUD dinâmica
- Efeitos visuais
- Sistema de vida

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Canvas API
